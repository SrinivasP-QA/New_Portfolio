# Srinivas P — Portfolio

## Upload to GitHub (Step by Step)

1. Go to github.com → Create a **New Repository** (name it anything e.g. `my-portfolio`)
2. Click **"uploading an existing file"**
3. **Drag and drop ALL these files** (select all files + assets folder):
   - index.html
   - style.css
   - script.js
   - assets/resume.pdf
4. Click **Commit changes**

## Deploy on Vercel

1. Go to vercel.com → Sign in with GitHub
2. Click **Add New Project** → Select your repo
3. **Root Directory** → leave it **EMPTY** (blank)
4. Click **Deploy** → ✅ Live!
